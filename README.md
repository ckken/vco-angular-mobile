## 集成框架明细 ##

`bouncefix.js` IOS 修复拖动bug框架

`boilerplate` html5 resset

`hammer.js` 移动设备手势操作

## Angular 模块 ##

`angular-mobile-nav` 移动设备切换效果模块

`angular-hammer` hammer 模块

`ng-mobile-menu` 菜单切换模块
